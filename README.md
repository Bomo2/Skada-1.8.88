# Skada for 3.3.5a

**Skada** is an addon that displays real-time combat statistics directly in the game. It provides a quick and easy way to track damage, healing, and overall performance for each player during a fight.


## What's new in this version ?

Damage module : (More oriented for Icecrown Citadel and Ruby Sanctum)

  - No more headaches trying to choose between Damage, Useful Damage, or Enemy Damage Taken on each encounter : Damage module is now all you need to accurately display DPS in conjunction with UwU Logs.
  - No more worries when a player encounters a mob in combat before pulling the boss (Critters, Dig Rat etc.).

Ennemy Damage Taken :

  - Added `Phase 2 - Pushed` unit to see which player(s) didn't stopped between 37-35% on Syndragosa encounter.
  - Added `Tombs - Pushed`   unit to see which player(s) didn't stopped on the last 5% tombs health on Syndragosa encounter.
  - Improved `Val'kyr Shadowguard` to get with precision the useful damage and in the same time you still have `Valkyrs overkilling` separated in another unit (for Lich King encounter).

Nicknames module :

 - Increased character length from 12 to 21.
 - You can add space in your Nickname.
 - Reloading or Logging off still displaying your Nickname on Skada.

   
## Installation

1. Download the [latest version](https://github.com/Bomo2/Skada-1.8.88/archive/refs/heads/main.zip) and dezip it.
      
2. Place `Skada` inside your `Interface/AddOns` directory.
   
3. Accept all changes if Windows asks you to.
